This is a C implementation of the shell using commands like execvp, getpid, pid, uname, getcwd, chdir, fork, strtok, signal, setbuf, getline, atoi etc..

The code supports ";" between commands and also handles extra spaces.
The code supports ">" as many time
The code supports ">>" as many times
The code supports "|" ones

The user defined commands implemented : jobs, kjob, fg, overkill, quit

The code can be run using gcc .
